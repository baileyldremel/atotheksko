Sketching out a typeface from my prototype

I felt the binary code idea was strong so I sketched out the full set of glyphs for the typeface using the same rules as my inital prototype. Some of the letters looked really cool and funky because I had to keep with the rules I made, but the set looked really good for something to start with. Some of the glyphs look similar to other glyphs or characters (the upper case S looked like a 5 originally), so I could probably work on that when I transition this over to the digital space.
