Random Arrays

Working with arrays, storing values and banging heads.

I was familiar with from my Year 12 programming class. I had the goal of each character being a different colour. I initially tried to do this by changing the colours when the mouse is pressed, but that would change the colour of all the letters, which was cool but not what I wanted. I then put the code in the function wiggle() but it ended up that every frame the colours of the letters would change. Although this was unintended, I thought it was cool and could be used as a glitch effect for my type sampler.
