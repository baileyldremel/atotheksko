Easter

Adding some little treats within my sampler

As I had gotten most of the basic functions working for my sampler, I wanted to experiment with adding some easter eggs in the sampler. I started out with adding a song called ‘hello world’ by Louie Zong and the Konami code. I wanted to add eggs that relate back to computers or code in some way as that was the purpose of the typeface. That’s why I chose these two because A. Hello World is a common bit of code that most programmers learn when they start programming and B. The Konami code is one of the most famous easter eggs in video gaming.
